# F&B Operations Agent API

AI-powered staffing forecasting for restaurants.

## Endpoints

- `GET /` - API info
- `GET /health` - Health check
- `GET /docs` - Swagger documentation
- `POST /predict` - Generate staffing prediction

## Tech Stack

- FastAPI + Uvicorn
- Claude AI (Anthropic)
- Python 3.11
